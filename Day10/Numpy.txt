Data Types Supported by Numpy
-----------------------------

1. int8, int16, int32, int64

2. uint8, uint16, uint32, uint64 - unsigned integer with different bit size

3. float32, floart64

4. complex64, complex128



Attributes of Numpy
-------------------
ndim		- returns the dimension of the array

dtype		- returns the data type of the elements

size		- returns the no of elements in the array

shape 		- returns the size of the array in each dimension

itemsize	- returns the size (in bytes) of each element in the 
		  array

data		- returns the buffer containing actual elements of the 
		  array in the memory


IO functions related to numpy

save()		- saves and array to a binary file in the numpy format

load()		- loads the data from binary file in the numpy

savetxt()	- saves the array to a text file to a specific format

loadtxt()	- load data from a text file
